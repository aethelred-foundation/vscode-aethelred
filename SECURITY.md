# Security Policy

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues or pull requests.

Send reports to **security@aethelred.io** with:

- affected extension version
- impacted operating system and VS Code version
- proof of concept or reproduction steps
- impact summary
- any relevant logs or screenshots

If the issue also affects the protocol or CLI, include the related repository, branch, tag, or release artifact.

## Scope

This repository covers the Aethelred VS Code extension source, packaging, and release metadata.

Protocol vulnerabilities should continue to follow the main Aethelred security program documented at:

- https://github.com/aethelred-foundation/aethelred/blob/main/SECURITY.md
